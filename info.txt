c.) pendahuluan 
1. C++ adalah bahasa pemrograman yang populer.
2. C++ digunakan untuk membuat program komputer, dan merupakan salah satu bahasa yang paling banyak digunakan dalam pengembangan game.
3. C++ dikembangkan sebagai perluasan dari C , dan kedua bahasa tersebut memiliki sintaksis yang hampir sama.
4. C++ adalah bahasa berorientasi objek dan beberapa konsep mungkin baru.

d.) Apa itu C++?
1. C++ adalah bahasa lintas-platform yang dapat digunakan untuk membuat aplikasi berkinerja tinggi.
2. C++ dikembangkan oleh Bjarne Stroustrup, sebagai perluasan dari bahasa C.
3. C++ memberi programmer kontrol tingkat tinggi atas sumber daya sistem dan memori.
4. Bahasa ini diperbarui 4 kali secara besar-besaran pada tahun 2011, 2014, 2017, dan 2020 menjadi C++11, C++14, C++17, C++20.

e.) Mengapa Menggunakan C++
1. C++ adalah salah satu bahasa pemrograman paling populer di dunia.
2. C++ dapat ditemukan dalam sistem operasi masa kini, Antarmuka Pengguna Grafis, dan sistem tertanam.
3. C++ adalah bahasa pemrograman berorientasi objek yang memberikan struktur yang jelas pada program dan memungkinkan kode untuk digunakan kembali, sehingga menurunkan biaya pengembangan.
4. C++ bersifat portabel dan dapat digunakan untuk mengembangkan aplikasi yang dapat diadaptasi ke berbagai platform.
5. C++ menyenangkan dan mudah dipelajari!
6. Karena C++ dekat dengan C , C#, dan Java , hal ini memudahkan programmer untuk beralih ke C++ atau sebaliknya.

f.) Perbedaan antara C dan C++
C++ dikembangkan sebagai perluasan dari C , dan kedua bahasa tersebut memiliki sintaksis yang hampir sama.
Perbedaan utama antara C dan C++ adalah bahwa C++ mendukung kelas dan objek, sedangkan C tidak.

g.) Untuk mulai menggunakan C++, Anda memerlukan dua hal:
1. Editor teks, seperti Notepad, untuk menulis kode C++
2. Kompiler, seperti GCC, untuk menerjemahkan kode C++ ke dalam bahasa yang dapat dimengerti oleh komputer.

h.) catatan 
1. Jangan khawatir jika Anda tidak mengerti cara #include <iostream> kerjanya using namespace std. Anggap saja itu sebagai sesuatu yang (hampir) selalu muncul dalam program Anda.
2. Setiap pernyataan C++ diakhiri dengan titik koma (;)
3. Isinya pada int main() juga dapat ditulis sebagai:
int main () { cout << "Hello World! "; return 0; }

Ingat: Kompiler mengabaikan spasi. Namun, beberapa baris membuat kode lebih mudah dibaca.

4. Program komputer adalah daftar "instruksi" yang akan "dieksekusi" oleh komputer.
Dalam bahasa pemrograman, instruksi pemrograman ini disebut pernyataan atau statement.

i.) baris baru
Keduanya \n dan endl digunakan untuk memutus baris. Namun, \n yang paling banyak digunakan adalah.

- Tapi apa \n sebenarnya?
Karakter baris baru (\n) disebut urutan escape, dan memaksa kursor untuk mengubah posisinya ke awal baris berikutnya di layar. Ini menghasilkan baris baru.

j.) Anda telah mempelajari bahwa cout digunakan untuk mengeluarkan (mencetak) nilai. dan kita akan menggunakan cin untuk mendapatkan masukan pengguna.
catatan :
1. cout diucapkan "see-out". Digunakan untuk output, dan menggunakan operator penyisipan (<<)
2. cin diucapkan "see-in". Digunakan untuk input, dan menggunakan operator ekstraksi (>>)
